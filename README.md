# File Manager - Pilot Program

Sistema de processamento e análise de pipelines de logística otimizado para alta performance (20.000+ linhas).

## Repositório Oficial
[https://github.com/zEduardoSSilva/file-manager](https://github.com/zEduardoSSilva/file-manager)

## Tecnologias
- **Next.js 15 (App Router)**
- **Genkit (AI Integration)**
- **Firebase (Firestore & App Hosting)**
- **ShadCN UI & Tailwind CSS**
- **SheetJS (XLSX)** para processamento massivo de dados

## Diferenciais
- **Pipeline Performaxxi Unificado**: Processamento linear que filtra automaticamente rotas em status `STANDBY`, permitindo analisar 20.000+ linhas em poucos segundos.
- **Bonificação Proporcional**: Cálculo automatizado baseado em 4 critérios fundamentais (Raio, SLA, Tempo e Sequência).
- **Consolidação Modular**: Sistemas integrados para CCO, Coordenadores, Faturistas e Roadshow.
