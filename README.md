
# vFleet Studio - File Manager

Sistema de processamento e análise de pipelines de logística.

## Repositório
[https://github.com/zEduardoSSilva/file-manager](https://github.com/zEduardoSSilva/file-manager)

## Tecnologias
- Next.js 15
- Genkit (IA)
- Firebase (Hosting & Firestore)
- ShadCN UI & Tailwind CSS

## Configuração do Pipeline
O sistema utiliza Server Actions otimizadas para processar grandes volumes de dados (20k+ linhas) em conformidade com as regras de bonificação proporcional de 4 critérios.
